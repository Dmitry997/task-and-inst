Тестовое задание для Junior DevOPS

Необходимо создать простую статическую html страницу index.html, содержащую ваши ФИО. Залить на любой общедоступный git репозиторий (github, bitbucket). Далее необходимо написать докерфайл, который создает образ на базе nginx с вашей страницей. Данный образ необходимо загрузить на dockerhub в вашу учетную запись. 
Если у вас нет каких либо учетных записей (github, bitbucket, dockerhub), то их необходимо создать.
Далее вашу html страницу необходимо запустить через helm в среде kubernetes. Для этого локально необходимо развернуть minikube, создать helm chart, заменить в нем образ на тот, который создали не предыдущем этапе и задеплоить через команду “helm upgrade –install”. 

Результатом выполнения задания является:
1.	Ссылка на репозиторий с докерфайлом, html файлом, helm чартом
2.	Ссылка на докерхаб где лежит ваш образ
3.	Демонстрация деплоя через helm на локальном ПК
4.	Открытая в браузере ваша html страница, запущенная в среде kubernetes
